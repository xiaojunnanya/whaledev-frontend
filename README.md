## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run dev

# watch mode
$ npm run preview
```



1、创建应用，在应用中创建页面，可以选择创建antd，show，还是服务编排
2、project/:id/page/:id  获取页面信息
3、页面支持复制，项目支持复制、删除、修改信息
